# Australia Map Coloring (CSP)

## Description
This project implements a solution to the classic Map Coloring problem using a Constraint Satisfaction Problem (CSP) approach. It colors the seven principal states and territories of Australia—Western Australia (WA), Northern Territory (NT), Queensland (Q), South Australia (SA), New South Wales (NSW), Victoria (V), and Tasmania (T)—ensuring that no two adjacent regions share the same color. 

The algorithm utilizes a simple backtracking search and requires only three colors (Red, Green, Blue) to successfully satisfy all adjacency constraints.

## Requirements
* Python 3.x
* No external libraries or dependencies are required.

## How to Run
1. Ensure you have Python installed on your system.
2. Save the `map_coloring.py` file to your local machine.
3. Open a terminal or command prompt.
4. Navigate to the directory where the file is saved.
5. Run the script using the following command:
   ```bash
   python australia-map-coloring-csp.py
   ```
